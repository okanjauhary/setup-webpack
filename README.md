# Setup Webpack

## Project setup
```
npm install
```
or
```
yarn install
```

### Compiles and hot-reloads for development
```
npm run start
```
```
yarn start
```

### Compiles and minifies for production
```
npm run build
```
```
yarn build
```

### Run your tests
```
npm run test
```

### Complete Documentation
See [Webpack](https://webpack.js.org/concepts/).
